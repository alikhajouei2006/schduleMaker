📅 University Schedule Builder

A simple yet powerful desktop application for creating and managing weekly university schedules, built with C# WinForms.
The app helps students generate all possible valid timetable combinations based on courses, instructors, and available times, allowing them to choose or save the best option.


---

✨ Features

✅ Add courses with multiple instructors and different time slots

✅ Automatically generate all valid, non-conflicting schedules

✅ Visual display of schedules in a weekly grid format

✅ Clean and easy-to-use interface



---

🛠️ Tech Stack

Language: C#

Framework: .NET WinForms
